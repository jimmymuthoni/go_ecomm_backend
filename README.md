### Go E-Commerce Backend API

A scalable and modular E-commerce Backend API built with Go (Golang) and MongoDB, designed to handle core e-commerce functionalities including user management, product management, store operations, and admin controls.

This backend is suitable for integration with web or mobile frontends (React, Next.js, Flutter, Android, etc.).

#### User Management
- User registration and login
- Secure authentication (JWT-based)
- Role-based access control (User / Admin)
- Profile management

#### Store Management
- Create and manage stores
- Assign store owners/admins
- Store-level product organization

#### Product Management
- Add, update, delete products
- Product categories
- Product pricing and stock management
- Fetch products by store or category

#### Admin Features
- Admin authentication
- Manage users and stores
- Approve or suspend stores
- System-wide product moderation

#### Database
- MongoDB as the primary datastore
- Well-structured collections
- Indexed fields for performance
